# Hardware Mecánico de SmartJoint

Esta carpeta contiene los planos y archivos de diseño mecánico de la articulación.

## Contenido esperado

- `stator.step` - Carcasa fija (estator).
- `rotor.step` - Eje giratorio (rotor).
- `housing.step` - Carcasa exterior.
- `assembly.step` - Conjunto ensamblado.
- `STL/` - Archivos para impresión 3D (versión de prototipo).

## Estado

- [ ] Diseño mecánico en CAD (pendiente).
- [ ] Archivos STEP para fabricación (pendiente).
- [ ] Archivos STL para impresión 3D (pendiente).

## Nota

El diseño mecánico no está disponible todavía. Se publicará cuando se complete el prototipo funcional. Mientras tanto, puedes usar las especificaciones dimensionales del `docs/SmartJoint_Specification.md` para crear tu propio diseño.
