# Ejemplos de software para SmartJoint

Esta carpeta contiene ejemplos de código para controlar la articulación desde una computadora.

## Contenido esperado

- `python/basic_control.py` - Conexión por UART (fibra o RS-485) y envío de comandos básicos.
- `python/position_mode.py` - Ejemplo de control en modo posición.
- `python/velocity_mode.py` - Ejemplo de control en modo velocidad.
- `python/torque_mode.py` - Ejemplo de control en modo par.
- `cpp/basic_control.cpp` - Ejemplo en C++ (para sistemas embebidos).
